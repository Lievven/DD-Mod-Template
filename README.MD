You can find the exact Godot version used to create Dungeondraft as of Dungeondraft version 1.2.0.1 here: https://godotengine.org/download/archive/3.4.2-stable/
You can find the Modding API documentation here: https://megasploot.github.io/DungeondraftModdingAPI/

If you want to employ the _Lib API (heavily recommended), you can find the most up-to-date version here: https://github.com/CreepyCre/_Lib
And the documentation to the _Lib API can be found here: https://creepycre.github.io/_Lib/

You **must** change the .ddmod file to values for your own mod, otherwise this **will** cause issues with Dungeondraft from just being in a user's mod folder.
